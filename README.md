# RecipeHub API

API REST para gerenciamento de receitas e categorias, desenvolvida em PHP puro.

## Tecnologias

- PHP 8+
- PostgreSQL
- Composer (autoload PSR-4)

## Instalação

```bash
composer install
```

## Configuração

Edite o arquivo `config/database.php` com as credenciais do seu banco PostgreSQL.

## Executando

```bash
php -S localhost:8000 -t public
```
